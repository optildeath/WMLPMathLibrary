Welcome to my Assessment ADGP_110

Here i have conducted a series of test off Ben Odom OpenGL engine with my own Vector Library.

The program is split into three classes, one set for 2D Vectors, 3D Vectors and Vector 4 . When a variable is declared by ether Vector2<> or Vector3<> the values set in the main can then be Added, Subtracts, give the Magnitude, Normalizer , Dot Product Cross Product. Linear Interpolation, That or the decimal value of a Color Hexadecimal .
